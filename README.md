# TodoApp

## Login 
Usernames and passwords are in https://dummyjson.com/users.
Both of fields are required.

![todo-app-login](https://user-images.githubusercontent.com/79408397/195543249-62711d0e-8b95-43a5-8d65-432f76bc39dc.png)

## Todo-list
You can swap bettween Todos and Users using top buttons.
You can delete, update and add new todos using fake APIs.

![todo-list](https://user-images.githubusercontent.com/79408397/195545149-b07e4491-f1b7-4311-9271-e5b69197a5fe.png)

## Users-list
List of users from https://dummyjson.com/users with pagination.
You can see the additional information about user, just click field with this user.

![users-list](https://user-images.githubusercontent.com/79408397/195545125-5de29a35-575c-41da-8ddb-3d4fd282dcff.png)

## User-info
Additional info about user.
